# 💻 MacOS Clone Portfolio Website

A visually rich and interactive **MacOS-inspired personal portfolio website** built to showcase my projects, skills, and personality in a unique and engaging way. This project replicates the look and feel of a real MacOS desktop environment while integrating modern web features.

---

## 🚀 Overview

This portfolio is not just a static website — it’s a **fully interactive desktop-like experience** where users can explore different sections through apps, just like on a Mac.

The goal of this project is to:

* Showcase my technical skills creatively
* Deliver an immersive and engaging user experience
* Blend UI/UX design with frontend development

---

🔗 https://mac-os-portfolio-eosin.vercel.app/

## ✨ Features

### 🧑‍💻 Desktop Interface

* MacOS-style UI with dock and app icons
* Window-based navigation system
* Smooth animations and transitions

---

### 📁 Applications

#### 📂 GitHub (Projects Viewer)

* Displays my projects with descriptions
* Acts as a dynamic portfolio section

---

#### 📝 Notes (About Me)

* Technical introduction
* Skills overview:

  * Programming languages
  * Frameworks & libraries
  * Tools & technologies

---

#### 📄 Resume (PDF Viewer)

* Embedded resume viewer
* Option to view or download CV

---

#### 📅 Calendar

* Redirects to Google Calendar
* Helps in scheduling and availability

---

#### 🎵 Spotify Integration

* Displays my favorite playlist
* Adds a personal and interactive touch

---

#### 📧 Email

* Opens default mail client
* Enables direct communication

---

#### 🔗 Links

* Quick navigation to:

  * GitHub profile
  * Other important platforms

---

#### 🖥️ CLI (Terminal Emulator)

* Built using `react-console-emulator`
* Simulates a terminal interface
* Supports custom/dummy commands

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** SCSS (Sass)
* **Libraries & Tools:**

  * react-console-emulator
  * Spotify Web API
  * Modern browser APIs

---

## 📦 Installation

```bash id="install001"
# Clone the repository
git clone https://github.com/CodeMonk21/Mac_OS_Portfolio.git

# Navigate into the project
cd Mac_OS_Portfolio

# Install dependencies
npm install

# Start the development server
npm start
```
